# Frontend Design Summary

Dokumen ini adalah ringkasan cepat arahan desain frontend.

## Referensi Desain

1. Gambar 1 menjadi acuan utama untuk landing page dan halaman publik.
2. Gambar 2 menjadi acuan utama untuk dashboard admin dan superadmin.

## Konsep Desain

Website menggunakan kombinasi:

1. Dark futuristic landing page.
2. Clean dark productivity dashboard.
3. Typography konsisten.
4. Dark-only mode.
5. Animasi halus.
6. Card-based layout.
7. Glassmorphism untuk public page.
8. Sidebar dashboard untuk admin dan superadmin.

## Mode Tampilan

- Landing page default: dark mode.
- Public page default: dark mode.
- User dashboard default: dark mode.
- Admin dashboard default: dark mode.
- Superadmin dashboard default: dark mode.

## Teknologi Frontend

- HTML native.
- CSS native.
- JavaScript native.
- Tidak menggunakan framework frontend pada versi awal.

## Prinsip Utama

1. Desain harus profesional.
2. Tampilan tidak boleh seperti template AI generik.
3. Warna, font, spacing, dan animasi harus konsisten.
4. Dashboard harus mudah digunakan.
5. Landing page harus terlihat premium dan modern.
