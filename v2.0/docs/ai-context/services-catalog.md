# Services Catalog

## Gambaran Umum

Sistem menyediakan layanan konsultasi teknologi yang dibagi ke dalam kategori layanan utama dan sublayanan. Pengguna memilih sublayanan tertentu sebelum melakukan pembayaran dan memulai konsultasi.

## Kategori Layanan Utama

Pada versi awal, terdapat tiga kategori layanan utama:

1. Network Architecture
2. Database Architecture
3. Web Server & Virtualization

## 1. Network Architecture

Layanan ini berfokus pada konsultasi terkait perancangan, evaluasi, dan perbaikan arsitektur jaringan.

### Contoh Sub Layanan

1. Perancangan topologi jaringan.
2. Segmentasi jaringan.
3. Konfigurasi DNS internal.
4. Perancangan VPN atau private network.
5. Troubleshooting koneksi antarserver.
6. Konsultasi keamanan jaringan dasar.

## 2. Database Architecture

Layanan ini berfokus pada konsultasi terkait desain, struktur, dan pengelolaan database.

### Contoh Sub Layanan

1. Perancangan struktur database.
2. Normalisasi database.
3. Optimasi query dasar.
4. Relasi antar tabel.
5. Backup dan restore database.
6. Konsultasi keamanan database dasar.

## 3. Web Server & Virtualization

Layanan ini berfokus pada konsultasi terkait web server, deployment, virtualisasi, dan pengelolaan server.

### Contoh Sub Layanan

1. Instalasi dan konfigurasi web server.
2. Deployment website PHP.
3. Konfigurasi virtual host.
4. Pengelolaan server virtual.
5. Integrasi web server dengan database.
6. Troubleshooting error server.

## Data Sub Layanan

Setiap sublayanan minimal memiliki data:

1. Nama sublayanan.
2. Kategori layanan.
3. Deskripsi sublayanan.
4. Harga konsultasi.
5. Admin penanggung jawab.
6. Status aktif atau tidak aktif.

## Aturan Layanan

1. Pengguna harus memilih sublayanan sebelum melakukan pembayaran.
2. Percakapan konsultasi baru dapat dilakukan setelah pembayaran valid.
3. Harga konsultasi dapat diedit oleh admin yang berwenang atau superadmin.
4. Superadmin dapat menambah layanan baru jika sistem dikembangkan lebih lanjut.
5. Sublayanan yang tidak aktif tidak ditampilkan kepada pengguna.
