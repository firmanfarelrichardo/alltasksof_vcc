# Project Overview

## Nama Sistem

Zeta.

## Deskripsi Umum

Sistem ini merupakan website konsultasi teknologi berbasis PHP native. Website menyediakan layanan konsultasi yang dikelola oleh beberapa admin sesuai bidang kompetensi masing-masing. Pengguna dapat melihat landing page, melihat layanan, memilih sublayanan, melakukan pembayaran biaya konsultasi, dan melakukan percakapan konsultasi melalui fitur chat di dalam website.

## Tujuan Sistem

Tujuan sistem adalah menyediakan platform sederhana untuk menghubungkan pengguna yang membutuhkan konsultasi teknologi dengan admin atau konsultan yang memiliki kompetensi tertentu.

## Fokus Layanan

Pada versi awal, sistem memiliki tiga kategori layanan utama:

1. Network Architecture
2. Database Architecture
3. Web Server & Virtualization

Struktur layanan dibuat fleksibel agar pada masa depan superadmin dapat menambahkan admin baru atau layanan baru tanpa mengubah konsep utama sistem.

## Jenis Website

Sistem berbentuk website dengan landing page sebagai halaman utama. Website memiliki area pengguna, area admin, dan area superadmin.

## Ruang Lingkup Versi Awal

Fitur yang termasuk dalam versi awal:

1. Landing page.
2. Halaman daftar layanan dan sublayanan.
3. Detail layanan dan sublayanan.
4. Registrasi pengguna.
5. Approval akun pengguna oleh superadmin.
6. Login pengguna setelah akun disetujui.
7. Pemilihan sublayanan konsultasi.
8. Pembayaran biaya konsultasi.
9. Percakapan konsultasi melalui chat internal website.
10. Admin membalas pesan konsultasi.
11. Admin mengelola harga konsultasi.
12. Admin melakukan CRUD sublayanan konsultasi.
13. Admin melihat riwayat konsultasi.
14. Admin melihat data keuangan layanan.
15. Superadmin mengatur role admin.

## Batasan Versi Awal

Sistem dibuat sederhana terlebih dahulu. Tidak perlu menambahkan fitur tambahan di luar kebutuhan yang sudah disebutkan, seperti notifikasi eksternal, integrasi WhatsApp, rating, kalender booking, AI chatbot, multi-bahasa, atau fitur kompleks lain.
